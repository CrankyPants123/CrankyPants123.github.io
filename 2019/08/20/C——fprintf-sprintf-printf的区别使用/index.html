C——fprintf、sprintf、printf的区别使用

以上函数所需头文件均为stdio.h

fprintf

用于格式化信息输入到指定的文件流中。

sprintf

用于把结果输入到指定的字符串中。只能用stdout

printf

输出结果到屏幕

我觉得printf的本质上就是一个fprintf(stdout,...)。就像HR的本质就是一个复读机（狗头。(&)
