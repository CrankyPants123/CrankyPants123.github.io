写在博客伊始

阿里的轻量服务器已经租用了半年，却未曾好好利用过。偶尔连接上，在上边敲俩指令。想想也是可笑。

时常不能自律，萎靡在寝室打游戏三餐靠外卖活得像个寄生虫，却又常常因此坐立不安，焦躁且抑郁。

写了删掉，又写又删。

其实、、很早之前就了解了有自建博客这种玩意儿。不过却一直没有付诸实际行动——即使我的服务器基本已经闲置了半年，即使我的谷歌云开了一年。不过我并不后悔。后悔之于我能带来的既得利益不大还很消耗我的精力。

从开始建设到目前的三篇文章，耗时一周左右。比起我的朋友，进度远落后于他们，我还没申请域名，主题设置还有好多没有摸清，不过我并不着急。我喜欢慢工出细活，域名这种事应该是想到了好的点子再买。而不是草草了事。

孤独但不寂寞。我很喜欢孤独。能让人自省。“君子和而不同”，这种不同就是我孤独感的来源。由此导致我一直在深深的疑惑——这世上到底有没有和我能够契合的人？

之前做了十六型的人格测试，测试的结果显示我是主人公（ENFJ-A ）人格（我一直在深深怀疑这是不是利用巴纳姆效应的一个心理学骗局）。不过最后我还是选择相信，因为测试说这种人格的人在人群中只占2%。这样我这样一个很少去认同其他观点的人产生了深深的认同感。除此之外的结论：“天生的领导者”、“魅力四射”等等词汇对我来说实在可笑。

很晚了，不写了！回寝室了。

再见。
